# Settings

## General

### Startup

- [ ] Open previous windows and tabs
- [ ] Always check if Firefox Developer Edition is your default browser

### Tabs

- [ ] Ctrl+Tab cycles through tabs in recently used order
- [X] Open links in tabs instead of new windows
- [ ] When you open a link, image or media in a new tab, switch to it immediately
- [X] Ask before closing multiple tabs
- [X] Ask before quitting with Ctrl+Q
- [X] Show an image preview when you hover on a tab

### Appearance

- Change to dark theme

### Language

- [X] Use your operating system settings for "English (Finland)" to format dates, times, numbers, and measurements.

- [X] Always ask you where to save files

### Browsing

- [ ] Use autoscrolling
